Names.js

Bringing flexibility and clarity to JavaScript function calls.

What is it?

names.js augments the function prototype. Features include:

  • Named, unordered arguments
  • Default argument values

Installation

You only need names.js (or the minified version, names.min.js) so you can do one of the following:

  • Clone the entire repo as a submodule of your project

    $ git submodule add https://github.com/markstickley/names.js.git path/to/clone/to

  • Go to the project page and click on names.js or names.min.js and download to include anywhere in your project

  • Or you could click one of the handy links at the top of the page to download the whole lot.

Loading

names.js can be loaded with requirejs but if you are not using that you can just include it in a script tag.

Usage

Use names.js by calling Function.prototype.applyNamed on any function. It works just like Function.prototype.apply (docs) except you pass an object as the second value instead of an array.

Basic usage

function anyOldFunction(arg1, arg2, arg3) {
    return [arg1, arg2, arg3];
}

anyOldFunction.applyNamed(scope, {
    arg2: 'foo',
    arg1: 'bar',
    arg3: 'baz'
}); // ['bar', 'foo', 'baz']

Omitting an argument sets it to null 

function anyOldFunction(arg1, arg2, arg3) {
    return [arg1, arg2, arg3];
}

anyOldFunction.applyNamed(scope, {
    arg3: 'baz',
    arg2: 'foo'
}); // [null, 'foo', 'baz']

Adding default values to arguments

function anyOldFunction(arg1, arg2, arg3) {
    return [arg1, arg2, arg3];
}
anyOldFunction.__namesArgs = {
    defaults: {
        arg1: 'default value'
    }
};

anyOldFunction.applyNamed(scope, {
    arg3: 'baz',
    arg2: 'foo'
}); // ['default value', 'foo', 'baz']

Specifying the argument order

You normally won't need to bother with this as names.js calculates it for you.

function anyOldFunction(arg1, arg2, arg3) {
    ...
}
anyOldFunction.applyNamed(scope, {});

// When running anyOldFunction.applyNamed for the first time it generates this:

anyOldFunction.__namesArgs.args; // ['arg1', 'arg2', 'arg3']

However, if you want to change the order or save on cycles by specifying the arguments you can as follows:

function anyOldFunction(arg1, arg2, arg3) {
    return [arg1, arg2, arg3];
}
anyOldFunction.__namesArgs = {
    args: ['arg2', 'arg1', 'arg3']
};

anyOldFunction.applyNamed(scope, {
    arg1: 'foo',
    arg2: 'bar',
    arg3: 'baz'
}); // ['bar', 'foo', 'baz']
